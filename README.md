FactoJS is a **2D Factorio-like game** built with **Vite + TypeScript + Three.js**.  
It is an experimental sandbox where automation, logistics, and chaos collide.

---

## 🎮 About the Game

FactoJS is a factory-building and automation game inspired by titles like Factorio.  
Players can place machines, connect production lines, and build increasingly complex automated systems in a 2D world rendered using Three.js.

The game strongly focuses on **programmable factories**, where production logic can be controlled and extended through scripting or logic systems.

The goal is simple:  
👉 build efficient production chains  
👉 optimize resource flow  
👉 survive your own complexity  
👉 automate everything that can be automated

## 🚀 Tech Stack

- ⚡ Vite — fast development and bundling
- 🟦 TypeScript — type-safe game logic
- 🌐 Three.js — rendering 2D world via WebGL

## 🧠 Core Concept

FactoJS is built around the idea of **programmable factories**.

Instead of only placing buildings, players can:

* define production logic
* automate resource routing
* build self-optimizing systems
* create complex factory “programs” using in-game systems

Factories are not just structures — they behave like **systems you design and program (WIP)**.
